# vivibio

> **vivid** × **vivere** × **bio** — 鮮やかに生きる人の名刺サイト

vivibio は、個人の自己紹介を一枚のWebページに集約するためのサイトです。
SNSや発信メディアへのリンクを中心に据え、NFCカードに書き込んで名刺代わりに配布する使い方を想定しています。

[English README](./README.en.md)

---

## ✨ コンセプト

- **vivid** — 鮮やかに、印象に残るプロフィール
- **vivere** — ラテン語で「生きる」。今この瞬間の自分を伝える
- **bio** — 経歴・人物像をシンプルに

ポートフォリオサイトとは異なり、作品集ではなく**「人」そのものへのリンク集**として機能することを目指します。
プレーリーカードのように、NFCカードにサイトURLを書き込んで配布することで、紙の名刺に代わる新しい自己紹介体験を提供します。

## 📦 想定機能

- プロフィール表示（名前・肩書・アイコン・自己紹介文）
- SNSリンク集（X, GitHub, Zenn など）
- 発信記事へのリンク
- お仕事・コンタクト情報
- QRコード表示（その場でスキャンしてもらう用）
- ダーク / ライトテーマ切り替え
- 多言語対応（日本語 / 英語）

## 🚀 セットアップ

```bash

# 本番ビルド
npm run build
```

## 📡 配布方法

このサイトは NFC カード（プレーリーカード等）に URL を書き込んで配布する用途を想定しています。
カードをスマートフォンにかざすだけで vivibio が開き、相手はその場で SNS をフォローしたり連絡先を取得したりできます。
