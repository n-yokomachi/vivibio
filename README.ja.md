# vivibio

> **vivid** × **vivere** × **bio**

vivibio は、作成者のプロフィールやSNSなどのリンクを集約するためのサイトです。
NFCカードのデザインも含んでいます。

[English README](./README.md)

---

## 🛠️ 技術スタック

| カテゴリ | 採用技術 |
|---|---|
| フレームワーク | Astro |
| 言語 | TypeScript |
| ホスティング | Cloudflare Pages |


## 🚀 セットアップ

> プロジェクト初期化後に追記予定

```bash
# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build
```
